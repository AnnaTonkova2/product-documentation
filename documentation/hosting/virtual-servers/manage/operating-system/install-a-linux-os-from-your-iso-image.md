# Install a Linux OS from your ISO image

You can install an OS on your VPS for free. Exception: Windows. You can only buy Windows while ordering a server.

1\. Power down the virtual machine.

<img src="https://support.gcore.com/hc/article_attachments/115010032989/2017-10-04_20-18-26.png" alt="" width="70%">

2\. Go to "Disks".

<img src="https://support.gcore.com/hc/article_attachments/115009992725/2017-10-04_20-19-43.png" alt="" width="70%">

3\. Click **ISO**.

<img src="https://support.gcore.com/hc/article_attachments/115009992745/2017-10-04_20-20-21.png" alt="" width="70%">

4\. For the **Image name** menu, select your ISO. For the **Boot priority**, select **To the beginning** and click **OK.**

<img src="https://support.gcore.com/hc/article_attachments/115009992765/2017-10-04_20-21-16.png" alt="" width="70%">

5\. Make sure the disk image has appeared on the **Virtual disks** list.

<img src="https://support.gcore.com/hc/article_attachments/115009992785/2017-10-04_20-21-44.png" alt="" width="70%">

6\. Restart your VM.

<img src="https://support.gcore.com/hc/article_attachments/115009992805/2017-10-04_20-22-30.png" alt="" width="70%">

7\. Click **VNC**.

<img src="https://support.gcore.com/hc/article_attachments/115010033029/______________2017-08-28___3.56.20.png" alt="" width="70%">

You'll see the boot menu of your ISO.
